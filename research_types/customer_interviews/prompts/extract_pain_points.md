# Task
You are helping a Product Manager discover meaningful user pain points from a customer interview.

Extract up to 5 pain points that represent real user friction, inefficiency, confusion, or blockers.

## Instructions
For each pain point:
- describe the pain clearly
- identify the affected user type
- include direct quotes as evidence
- estimate severity and confidence
- avoid generic summaries
- avoid inventing any information not present in the transcript

Prefer pains that are:
- repeated
- operationally meaningful
- relevant to our product strategy
- likely to affect adoption, efficiency, or trust

Return valid JSON only.
